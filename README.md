## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/luffyy01/AI-IMAGE-GEN.git
cd AI-IMAGE-GEN
```

### 2. Install dependencies

```bash
npm install
# or
pip install -r requirements.txt
```

### 3. Run project

```bash
npm start
# or
python app.py
```
